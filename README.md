# Skyblue
## 概要
ビルの画像を共有するためのWebサイトです。Skylineの制作で超高層ビルに関する作品をまた作りたくなったので制作しました。

## 主な使用技術
- React
- TypeScript
- Tailwind CSS
- Firebase
- Google Maps Platform
